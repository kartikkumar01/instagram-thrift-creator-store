# Thrift & Handmade Store ER Diagram

This project contains an ER diagram for a thrift and handmade products store.

It includes:

* Users / customers
* Products
* Categories
* Colors
* Orders
* Transactions
* Order-Product junction table

The database supports:

* Multiple orders per customer
* Multiple products per order
* Thrifted and handmade products
* Product quantity, price, color, and category
* Payment status and shipping status
* Order history and customer details

The design is normalized and uses proper PK, FK, and relationships.
